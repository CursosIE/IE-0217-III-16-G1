Reconocimiento de genes en secuencias de ADN utilizando el algoritmo Aho-Corasick  V1.0
El programa toma una secuencia de nucleotidos de ADN y realiza una busqueda de genes (subsecuencia) y crea una lista a partir de un porcentaje especificado como minimo para hacer match. Desarrolladores: Giancarlo Marin, Robin Gonzalez, Luis Adrian Aguilar
